Fabric

Multi-channel transaction fabric of the internet

Hyperledger Fabric is an implementation of blockchain technology, leveraging familiar and proven technologies. It offers a modular architecture allowing pluggable implementations of various capabilities, such as smart contracts (chaincode), cryptography, consensus, ledger datastore and more. It leverages powerful container technology for its deployment and for providing the execution environment for the chaincode such that the network is protected from malicious code.

What differentiates Hyperledger Fabric?

Modular architecture Optimized for transaction performance and throughput
Built for the enterprise Permissioned network design
Pluggable consensus, identity, encryption, and data store Multi-channel communications
